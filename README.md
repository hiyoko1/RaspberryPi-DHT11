# RaspberryPi-DHT11

電子工作動画「ゆっくり電子工作【RaspberryPiと温度センサーでIoTシステムを作ってみる】」のその①、その②で使用したプログラムです。
30秒に一度、Ambientにデータを送信する仕組みなっています。
プログラムは自由に使っていただいて大丈夫ですが、何か問題点がございましたらご連絡ください。
